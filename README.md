# 💻 Repositório de Desenvolvimento de Sistemas (Web & PHP)

Este repositório armazena os exercícios, projetos e conteúdos práticos da disciplina de Desenvolvimento de Sistemas. O foco principal é o desenvolvimento Web back-end utilizando **PHP**, passando desde os conceitos estruturados básicos até a Programação Orientada a Objetos (POO) e a arquitetura estrutural MVC (Model-View-Controller).

## 🗂️ Estrutura do Repositório

Os materiais estão organizados de forma cronológica por aulas e por projetos práticos consolidados:

### 📚 Aulas
* **`aula1_13032026` a `aula2_atividade_20032026`:** Introdução ao PHP, criação e manipulação de formulários web (métodos GET e POST) e processamento básico de dados (ex: cálculo de médias).
* **`aula4_10042026` a `aula6_08052026`:** Aprofundamento na linguagem, modularização (utilização de componentes em PHP) e manipulação de ficheiros de texto (I/O).
* **`aula7_15052026` e `aula8_22052026`:** Introdução à **Programação Orientada a Objetos (POO)** em PHP (abstração de classes como `Cliente`, `Produto` e `Usuario`).

### 🚀 Projetos e Trabalhos
* **`cadastro-clientes`:** Sistema prático de cadastro aplicando a arquitetura **MVC** (Models, Views e Controllers) e estruturação de acesso a banco de dados.
* **`projeto_php`:** Projeto completo de um gestor de jogos utilizando o padrão MVC. Inclui as rotas, integração com banco de dados (acompanha o script `banco.sql`) e interface estilizada com Bootstrap.
* **`trabalho_produtos`:** Sistema desenvolvido para leitura, processamento e gravação em lote de dados de produtos a partir de ficheiros de texto (`produtos.txt`).
* **`docs`:** Exercícios complementares e implementações de Front-end envolvendo HTML, CSS e JavaScript.

## 🛠️ Tecnologias e Ferramentas Utilizadas

* **Linguagem Principal:** PHP
* **Front-end:** HTML5, CSS3, JavaScript e Bootstrap
* **Banco de Dados:** SQL (Script estrutural e queries de manipulação)
* **Paradigmas/Arquitetura:** Programação Orientada a Objetos (POO) e MVC
